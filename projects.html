Prototype Showcase

This is a modified prototype of Mission 02 from the Unity Learn Junior Programmer Pathway. It was a real challenge, but I'm very proud to have created a more user-friendly prototype. I modified the movement and camera, as well as how the bone shoots. I also added Game Over and Restart buttons. It is still a work in progress. The next steps are to add a 'Title' screen with instructions on how to play the game, as well as an option to return to the title screen and a pause button and a score for the fed dogs during gameplay. I will upload it once it is more complete.

Tools and skills practiced in this project: Unity v 6.3 | C#
