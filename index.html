Junkie Alchemist

Published

The jam was part of the Unity Game Developer Bootcamp by Generation Colombia.

  • Goal: Create a game within a strict time limit for the "You shouldn't mis those" jam hosted by Capybara Studios.
  • Description: In this game, you are an alchemist with an addiction. You need to fulfill clients' orders by mixing the necessary ingredients for potions.
